# EconoWorld - Financial Web Portal

**EconoWorld** is a web platform designed to provide a modern, secure, and high-performance interface for financial data management.

### 📅 Timeline
* **Status:** Completed (October 2025 – February 2026).

### 💻 Tech Stack
* **Frontend:** Angular (Modular and reactive architecture).
* **Backend:** **Firebase** (Firestore for real-time storage and Firebase Auth for security).
* **Integration:** Secure RESTful APIs and optimized database queries.

### 🔑 Key Features
* **Authentication:** Robust login and registration system powered by Firebase.
* **Performance:** Refactored relational structures to reduce latency and improve data retrieval.
* **UI/UX:** Dynamic and responsive portal integrated seamlessly with cloud services.
